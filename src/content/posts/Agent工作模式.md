| 模式                  | 核心特点          | 典型场景          |
| ------------------- | ------------- | ------------- |
| ReAct（Reason + Act） | 思考→行动→观察→再思考  | 通用 Agent、代码修复 |
| Plan-and-Execute    | 先整体规划，再按计划执行  | 长任务、项目开发      |
| Reflection          | 完成后自我检查并修正    | 提高代码质量、写作     |
| Multi-Agent         | 多个 Agent 分工协作 | 大型研发、复杂分析     |
| Human-in-the-loop   | 关键节点等待人工确认    | 高风险操作、生产环境    |
