| 对比项                  | InnoDB（默认） | MyISAM  | Memory          |
| -------------------- | ---------- | ------- | --------------- |
| 存储位置                 | 磁盘         | 磁盘      | 内存(RAM)         |
| 是否支持事务               | ✅ 支持       | ❌ 不支持   | ❌ 不支持           |
| 是否支持行锁               | ✅ 支持       | ❌ 只支持表锁 | ✅ 表锁            |
| 是否支持外键               | ✅ 支持       | ❌ 不支持   | ❌ 不支持           |
| 崩溃恢复(Crash Recovery) | ✅ 支持       | ❌ 不支持   | ❌ 重启数据丢失        |
| MVCC                 | ✅ 支持       | ❌ 不支持   | ❌ 不支持           |
| 默认索引                 | B+Tree     | B+Tree  | Hash（默认）、B+Tree |
| 全文索引                 | ✅ 支持(5.6+) | ✅ 支持    | ❌               |
| 查询速度                 | 快          | 很快      | 极快              |
| 写入速度                 | 较快         | 快       | 极快              |
| 重启后数据                | 保留         | 保留      | 全部丢失            |
| 默认引擎(MySQL8)         | ✅ 是        | ❌       | ❌               |
